全球领先的媒体技术供应商 Avid，如今发布了全新版本的业界领先数字音频工作站

Pro Tools 2018

Pro Tools 2018 增加的 EQ 缩略图

Pro Tools 是现在业界最开放、最高效的音频制作平台，Pro Tools 2018 在音乐制作上增加经验，拥有更快的制作流程，这些都是通过全新的轨道预设功能(Track Presets)实现的。其次还拥有许多创新性功能，如进一步提高 MIDI 编辑与播放列表的改进等。

Avid 对之前几个版本的用户反馈做出相应，为 2018 新版注入了更多音乐创作特点。让使用 Pro Tools 的作曲家、音乐人、制作人与录混音师专注于音乐创作过程，而不是将时间浪费在复杂的软件使用上。全新的回顾 MIDI 功能确保用户不会遗漏掉自己创作的精彩演奏片段，使用全新快捷键完成替换、修剪、编辑速度等动作，让音乐创作更简单更迅速。

“使用 Pro Tools 2018 的音乐家们可以迅速将创作灵感付诸于现实！” Avid 首席产品官 Dana Ruzicka 介绍到。“全新的 Pro Tools 可以给予有抱负的音乐人更大的创作空间，同时基于 Avid 产品生态系统，在合作创作方面也会更加流畅。”

全新 Pro Tools 2018 专注于音乐人的创作手法和灵感，准备更符合他们使用习惯的功能包括：

  • 轨道预设可以对轨道设置快速保存、编辑、找回你喜爱的效果连、工程预设等其他轨道设置。

  • 全新播放列表可以节省宝贵的屏幕空间，拥有全新波形图，可对片段进行编组。

  • 轻松将工程转换为云文件上传以便更快的进行合作。

  • 混音窗口加入 EQ 缩略图，更直观的展现轨道上的 EQ 插件参数。

更多详情请至 Avid 官网查看:www.avid.com/pro-tools
